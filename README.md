# UNO スコア

UNO の点数を管理するウェブアプリ。スマホのブラウザからすぐ使えます。

**https://shinichinishikawa.github.io/uno/**

## 使い方

1. 参加者の名前を入力して「ゲーム開始」
2. 「ゲームスタート」→ 勝った人を選ぶ（0点が入る）
3. 他の人の点数をタップして入力
4. 全員の点数が入ると「パンパカパーン！！」
5. 「もう一回」で次のラウンドへ

## 機能

- 合計点の低い順に自動ソート・順位表示
- LocalStorage でスコア永続化（リロードしても消えない）
- 途中から参加者追加も可能
- モバイル対応

## 技術

単一 HTML ファイル。ビルドステップなし、vanilla JS、Google Fonts CDN。
